# Console-Based Event Booking Portal

## Overview
This is a simple Java-based console application for booking events. It connects to a MySQL database using JDBC and allows users to register, log in, add events, view events, book them, and see their bookings.

## Modules
- **DatabaseConnection.java**: Establishes JDBC connection with MySQL.
- **User.java**: Manages user login and registration.
- **Event.java**: Allows admin to add new events and users to view them.
- **Booking.java**: Enables booking of events and viewing user bookings.
- **Main.java**: Acts as the entry point and main menu controller.
